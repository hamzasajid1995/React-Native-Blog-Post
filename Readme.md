## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/hamzasajid1995/React-Native-Blog-Post
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Start project
   ```sh
   yarn start
   yarn run ios
   yarn run android
   ```
